================= Mini-curso Docker - Otávio Kessler Ustra =================

1° Prática - Rodando Python no Docker
| Rode a imagem da nuvem do python usando: "docker run -it python bash"
| Dentro do bash do container, rode o interpretador do python usando: "python"
| Escreva um código em python. Ex print("Hello, container")
| Saia do interpretador: "exit()"
| Sai do bash do container: "exit"
